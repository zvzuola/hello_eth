### dapp 练手

### 项目搭建

```
mkdir hello_eth
npm i -D -g truffle 
truffle init
npm init -y
```

### 编译合约

```
truffle compile
```

### 部署合约
前往https://remix.ethereum.org进行部署，部署成功后点击detail获得部署的地址,项目中实例合约的时候需要用到

### metamask
安装metamask，https://metamask.io/，选择Ropsten Test Net，点击BUY获取ether

### 运行项目

```
npm i
npm run build
npm run server
```
